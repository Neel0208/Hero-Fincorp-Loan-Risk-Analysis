# Hero-Fincorp-Loan-Risk-Analysis
Data analysis project on loan risk and customer behavior using Python. Includes data cleaning, EDA, default risk analysis, customer segmentation, and business insights based on financial datasets.

# 📊 Loan Risk & Customer Behavior Analysis

## 📌 Overview
This project analyzes loan data to understand default risk, customer behavior, and business performance.

## 🛠 Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook

## 📊 Key Analysis
- Data cleaning & preprocessing
- Default risk analysis
- Customer segmentation
- Branch performance analysis
- EMI & transaction behavior
- Time-series trends

## 🔍 Key Insights
- Default risk is influenced by credit score, income, and EMI
- High-risk customers identified using segmentation
- Regional differences observed in performance and risk

## 🚀 Outcome
This project demonstrates data-driven decision-making for credit risk and financial analytics.
